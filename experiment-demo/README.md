# 物理实验不确定度计算工具

> 一款专业、高效的物理实验数据处理工具，基于Go语言开发，提供Web界面进行不确定度计算和结果导出。

## 功能特性

- 🧮 **自动计算**：自动计算平均值、标准偏差、A类不确定度、B类不确定度及合成不确定度
- 📊 **多组数据支持**：支持同时处理多组实验数据
- 📁 **结果导出**：自动将计算结果导出为CSV格式文件
- 🌐 **Web界面**：基于浏览器的交互式操作界面，无需安装额外软件
- 🎨 **开屏动画**：优雅的开屏封面设计，提升用户体验
- ⚡ **实时响应**：动态生成数据输入表单，操作便捷

## 技术栈

- **后端**：Go语言标准库（net/http、encoding/csv、text/template等）
- **前端**：原生HTML5 + CSS3 + JavaScript
- **数据格式**：CSV导出

## 快速开始

### 前提条件

- Go 1.18+ 环境

### 安装与运行

```bash
# 克隆仓库
git clone <repository-url>

# 进入项目目录
cd experiment-demo

# 运行程序
go run src/main/实验.go
```

程序启动后，打开浏览器访问：`http://localhost:8082`

## 使用方法

### 1. 启动程序

运行程序后，会自动启动Web服务器（默认端口8082）。

### 2. 进入主界面

按任意键跳过开屏封面，进入数据输入界面。

### 3. 输入实验参数

| 参数 | 说明 | 示例 |
|------|------|------|
| 实验数据的组数 | 需要计算的实验数据组数 | 3 |
| 每组的测量次数 | 每组实验的重复测量次数 | 6 |
| 仪器的误差限 (delta) | 测量仪器的最大允许误差 | 0.01 |

### 4. 输入测量数据

在动态生成的数据输入框中，输入每组实验的测量值，数值之间用空格分隔。

**示例**：
```
1.23 1.25 1.24 1.26 1.23 1.25
```

### 5. 计算与导出

点击"计算并导出结果"按钮，系统将自动：
- 计算各项统计参数
- 显示详细计算结果
- 导出CSV文件到 `./exports/` 目录

## 计算公式

### 平均值
```
L̄ = ΣLi / n
```

### 样本标准偏差
```
s = √[Σ(Li - L̄)² / (n-1)]
```

### A类不确定度
```
U_A = s / √n
```

### B类不确定度
```
U_B = Δ / (2√3)
```
其中 Δ 为仪器误差限

### 合成不确定度
```
U = √(U_A² + U_B²)
```

### 最终测量结果
```
结果 = L̄ ± U
```

## 开发指南

### 核心函数说明

```go
// 计算平均值
func average(arr []float64) float64

// 计算样本标准偏差
func S1(li []float64) float64

// 计算A类不确定度
func Uncertainty_A(li []float64) float64

// 计算B类不确定度
func Uncertainty_B(delta float64) float64

// 计算合成不确定度
func Uncertainty合成(A, B float64) float64

// 导出结果为CSV
func exportToCSV(results []Result, delta float64) (string, error)
```

### 数据结构

```go
type Result struct {
    GroupNum int       // 组号
    Data     []float64 // 原始测量数据
    Avg      float64   // 平均值
    StdDev   float64   // 标准偏差
    U_A      float64   // A类不确定度
    U_B      float64   // B类不确定度
    U        float64   // 合成不确定度
}
```

## 注意事项

1. **输入验证**：程序会对输入数据进行验证，确保数值格式正确
2. **数据完整性**：每组数据必须包含指定数量的测量值
3. **导出目录**：CSV文件会自动保存到 `./exports/` 目录，目录不存在时会自动创建
4. **文件命名**：导出文件按时间戳命名，格式为 `实验数据结果_YYYYMMDD_HHMMSS.csv`
